# Sistema Solar

Proyecto web interactivo sobre el sistema solar.

## Como verlo

Abri `index.html` en el navegador o publica el repositorio con GitHub Pages.

## Estructura

- `index.html` - pagina principal
- `audio/` - narraciones
- `videos/` - videos de planetas (comprimidos)
- `imagenes/` - imagenes estaticas

## Tamano

El proyecto esta optimizado para pesar menos de 25 MB y poder subirse a GitHub.
